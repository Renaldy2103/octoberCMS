Pengertian

October CMS merupakan sebuah CMS yang berbasiskan laravel framework. Hadir dengan struktur yang rapi dan menyajikan kemudahan dalam modofikasi-nya, bahkan sudah ada plugin yang memudahkah para pengguna October untuk membuat plugin sendiri (rainlab builder), sangat membantu para developer untuk membuat plugin sendiri dalam project-nya. Pada kesempatan ini akan dibahas bagaimana cara instalasi october menggunakan git, jadi kita akan clone repo dari october ini.

Sebelum masuk ke tahap instalasi, pastikan system ada memenuhi requirements berikut ini :

1. PHP version 5.5.9 or higher

2. PDO PHP Extension

3. cURL PHP Extension

4. OpenSSL PHP Extension

5. Mbstring PHP Library

6. ZipArchive PHP Library

7. GD PHP Library

Cara Dan Proses :

Informasi diatas berdasarkan situs resmi October CMS. Jika sudah memenuhi requirements diatas, pastikan di komputer anda sudah terinstal GIT dan composer, karena kita akan melakukan cloning dari github menggunakan GIT.

1. Jalankan perintah berikut di terminal / command prompt git clone https://github.com/octobercms/october.git.Tunggu sampai proses cloning selesai.

2. Setelah proses selesai, masuk ke directory yang hasil cloning tadi (dafault-nya foldernya adalah october), kemudian jalankan perintah composer update.

3. Jika proses update telah selesai, sekarang kita bisa melakukan instalasi dengan menggunakan perintah php artisan october:install.

~. Anda akan diminta untuk memilih database yang akan digunakan, kali ini kita akan menggunakan database MySQL.

~. Pilih host yang akan digunakan, default-nya adalah localhost, langsung tekan enter jika mau menggunakan default-nya.

~. Masukkan port MySQL, standartnya adalah 3306.

~. Kemudian masukkan nama database, pastikan sebelumnya anda sudah membuat database-nya.

~. Masukkan username dan password untuk login ke MySQL.

~. Sampai disini kita sudah selesai untuk konfigurasi database nya, selanjutanya kita akan diminta untuk memasukkan username, password, dan informasi lainnya untuk login ke admin panel nantinya.

~. Kemudian akan muncul konfigurasi untuk melakukan konfigurasi opsi lanjutan (advanced options) atau tidak, langsung enter saja, kita tidak perlu melakukan konfigurasi ini, kecuali anda sudah expert, :D

Kita sudah berhasil melakukan instalasi October CMS. Untuk melihat hasilnya anda bisa menjalalankan perintah php artisan serve. Kemudian buka di browser localhost:8000.
